JS Audio Tags
=====

JavaScript library to read tags from audio files.
Using HTML5 FileAPI.

Implemented features
--------------------

- Read ID3v1 from MP3 files (all versions: 1.1 and 1.1.1)
- Read ID3v2 from MP3 files (all versions: 2.2.0, 2.3.0, 2.4.0)
